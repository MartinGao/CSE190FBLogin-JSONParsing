CocoaPods

What is CocoaPods?

CocoaPods is the dependency manager for Swift and Objective-C Cocoa projects. It has over ten thousand libraries and can help you scale your projects elegantly.

Install CocoaPods

OpenTernimal , type

sudo gem install cocoapods

Create Podfile

Create a file named Podfile in your Xcode project ROOT directory

Example Podfile

platform :ios, '8.0'
use_frameworks!
pod 'Alamofire'
pod 'Facebook-iOS-SDK'

Install Dependencies

OpenTernimal , in your Xcode project ROOT directory , type

pod install

Usage

Use .xcworkspace since now

DO NOT Use .xcodeproj

Facebook Login

Setup Facebook Developer Account

  • Login to Facebook Developer Website Link

  • Nav Bar -> My Apps -> Add a New App -> iOS App

  • Enter your “Facebook App ID”

Install Facebook SDK

  • Manually Install - Please refer to Facebook Developer Website

  • CocoaPods Install - pod 'Facebook-iOS-SDK'

Setup Xcode Project

  • Modify info.plist

  • Enter the “Bundle Identifier” of your iOS Application at Facebook Developer Website

  • import Facebook header files in your Bridging-Header.h

    #import <FBSDKCoreKit/FBSDKCoreKit.h> #import <FBSDKLoginKit/FBSDKLoginKit.h> #import <FBSDKShareKit/FBSDKShareKit.h>

  • Edit AppDelegate.swift

    • Find function applicationDidBecomeActive

      Add -> FBSDKAppEvents.activateApp()

    • Find function “didFinishLaunchingWithOptions”

      Add -> FBSDKApplicationDelegate.sharedInstance().application(application, didFinishLaunchingWithOptions: launchOptions)

    • Create function func application(application: UIApplication, openURL url: NSURL, sourceApplication: String?, annotation: AnyObject) -> Bool

      Add -> return FBSDKApplicationDelegate.sharedInstance().application(application, openURL: url, sourceApplication: sourceApplication, annotation: annotation);”
  • In your ViewController.swift, add following lines into viewDidLoad() function

    let loginButton:FBSDKLoginButton = FBSDKLoginButton()
    loginButton.center = self.view.center;
    self.view.addSubview(loginButton)
